## 1. How to install the app 
  a) Unzip the file 'JinukSun-hatchways-assessment-FrontEnd.zip'  
  b) Change you directory to 'hatchways-assessment' inside the unzipped directory  
  c) npm install axios  
        or  
       yarn add axios  
 
## 2. Run the app 
  a) npm start  
        or  
       yarn start
